# GoChat – Real-Time Chat Application

A full-stack real-time chat application built using **Golang, WebSockets, PostgreSQL, and Docker**, featuring secure JWT authentication, persistent storage, and a modern Spider-Man themed UI.

---

## Features

- **JWT-based Authentication**
  - User registration and login
  - Secure password hashing using bcrypt
  - Stateless session management

- **Real-Time Messaging**
  - WebSocket-based communication
  - Concurrent multi-user chat
  - Instant message broadcasting

- **PostgreSQL Integration**
  - Persistent user and chat data storage
  - Relational schema design
  - Data survives server restarts

- **Dockerized Environment**
  - Multi-container setup using Docker Compose
  - Backend and PostgreSQL services
  - Persistent database volume

- **Modern Frontend**
  - Built with HTML, CSS, and JavaScript
  - Spider-Man inspired UI
  - Responsive and interactive chat experience

---

## Tech Stack

- **Backend:** Go (Golang), net/http, WebSockets
- **Authentication:** JWT, bcrypt
- **Database:** PostgreSQL
- **Frontend:** HTML, CSS, JavaScript
- **Infrastructure:** Docker, Docker Compose
